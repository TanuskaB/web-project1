Developers

Tanuska Biswakarma and Caira Major

Recipe Studio was developed by Tanuska Biswakarma and Caira Major, two dedicated developers who collaborated to create a clean, user-friendly, and well-structured recipe management platform. Together, they focused on building an intuitive interface combined with organized and scalable data handling to ensure a smooth and enjoyable user experience.

Tanuska contributed to the overall application structure, data organization using JSON formatting, and implementation of core functionalities, ensuring the system remains efficient and maintainable. Caira played a key role in refining the user interface design, enhancing usability, and supporting feature development to create a visually appealing and responsive platform.

Through teamwork, attention to detail, and a shared commitment to quality, they successfully developed Recipe Studio as a practical and scalable web application that can be expanded with additional features in the future.
